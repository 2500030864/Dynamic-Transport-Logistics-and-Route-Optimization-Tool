# 🚀 Dynamic Route Optimizer

A logistics route optimization tool using Dijkstra’s algorithm with dynamic traffic simulation.

## 📌 Features
- Graph-based routing
- CSV-based data input
- Dynamic traffic simulation
- Shortest path calculation

## ▶️ Run
python main.py

## 📁 Data Format (routes.csv)
Each row: source,destination,distance

Example:
A,B,4
A,C,2
C,D,3
